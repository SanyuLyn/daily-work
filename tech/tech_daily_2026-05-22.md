### [Flutter 鸿蒙 2026 路线发布，加速同步官方生态，进一步优化体验](https://zhuanlan.zhihu.com/p/2016571565735682780)
**摘要**：Flutter-OH 2026 年路线图正式发布，核心目标是缩短版本落差，适配周期从过去的 7 个月以上压缩到约 4 个月内，计划按季度跟进官方 Flutter 发布节奏。Flutter SIG 将主导维护工作，规划至少 200 个高优先级三方库的鸿蒙适配，涵盖网络、数据库、图片处理、音视频等常见领域。Impeller 渲染后端对鸿蒙 Vulkan 驱动的适配逐步稳定，相比 Skia 在帧率稳定性上有显著提升，整体鸿蒙生态建设进入提速阶段。

### [Flutter Web 正式移除 HTML renderer，只支持 CanvasKit 和 SkWasm](https://zhuanlan.zhihu.com/p/11809045697)
**摘要**：Flutter 官方正式弃用并移除 HTML renderer，Flutter Web 将只保留基于 WebGL 的 CanvasKit 和 SkWasm 两种渲染器。HTML renderer 因性能不足、维护成本高、图形表现力受限而被淘汰。SkWasm 在 WebAssembly 模式下启动时间和帧率表现更优，CanvasKit 作为通用 fallback。开发者需尽快迁移，移除对 `--web-renderer html` 参数的使用，现有使用 HTML renderer 的项目在未来版本中将无法正常构建。

### [Flutter-Web 从 0 到部署上线（实践+埋坑）](https://zhuanlan.zhihu.com/p/1938276051617845509)
**摘要**：文章系统梳理了 Flutter Web 从工程创建到 CDN 部署的完整实践流程，涵盖环境配置、渲染器选择（CanvasKit vs SkWasm）的利弊权衡、包体积优化以及常见踩坑点。对于已有 Flutter 移动端经验的开发者，Flutter Web 可做到零额外学习成本的代码复用，但需注意 CanvasKit 模式下包体积偏大的问题，建议结合 CDN 加速与懒加载策略优化首屏性能。

### [Flutter 3.41 发布，鸿蒙如何选择？](https://zhuanlan.zhihu.com/p/2005202488488378695)
**摘要**：Flutter 官方已发布 3.41 版本并公布季度版本节奏，而鸿蒙社区面临版本跟进与差异化适配的双重挑战。文章分析了 Flutter-OH 与官方 Flutter 3.41 的主要差异，以及鸿蒙开发者在选择 Flutter 版本时需要权衡的性能、生态兼容性和维护窗口等核心因素，为正在从事 Flutter 鸿蒙开发的团队提供了决策参考。

### [鸿蒙版 Flutter 三方库适配案例【screen_brightness】](https://zhuanlan.zhihu.com/p/29600343631)
**摘要**：以 screen_brightness 插件为例，详细介绍了将 Flutter 三方库适配到鸿蒙平台的完整流程，覆盖 HarmonyOS 平台特有的 API 调用方式、DevEco Studio 工程结构配置，以及如何通过 Flutter Plugin 机制在 Dart 层与鸿蒙原生层之间建立通信桥接。该案例对正在推进 Flutter 鸿蒙生态建设的开发者具有重要的实操参考价值。

### [Android Gradle Plugin 9.0 发布，为什么这会是个史诗级大坑版本](https://zhuanlan.zhihu.com/p/1997655149036995133)

### [如何看待谷歌逐步收紧安卓应用侧载计划？](https://www.zhihu.com/question/2019018192186557023)
**摘要**：Google 通过"高级侧载流程"逐步收紧 Android 应用侧载权限，新流程要求用户完成六步验证并等待 24 小时强制期。知乎社区对此看法分歧：部分人认为这是以安全为名限制开放性，向 iOS 封闭生态靠拢；另一部分认为有助于遏制恶意软件传播。核心争议在于 Google 是否在借安全名义构筑应用生态壁垒，压缩独立开发者和第三方应用市场的生存空间。

### [Google 推出耗电应用标记｜Android 出海政策周报](https://zhuanlan.zhihu.com/p/1982391265216701342)
**摘要**：Google 宣布自 2026 年起将在 Google Play 商店中为高耗电应用添加醒目标签，帮助用户在安装前了解应用的电量消耗情况。本期周报同时汇总了近期 Android 出海开发者需关注的政策变动，包括数据隐私合规要求更新、目标国家应用审核新规等，是出海 Android 开发团队的重要政策导航参考。
