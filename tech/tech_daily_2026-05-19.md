## Flutter

### [Flutter Web 正式移除 HTML renderer，只支持 CanvasKit 和 SkWasm](https://zhuanlan.zhihu.com/p/11809045697)
**摘要**：Flutter 官方宣布正式从 Flutter Web 中移除 HTML renderer，未来只保留 CanvasKit 和 SkWasm 两种渲染后端。CanvasKit 基于 Skia 图形库通过 WebAssembly 运行，可实现与移动端高度一致的渲染效果；SkWasm 则是更轻量的 WASM 渲染方案，启动更快、包体更小。此次变更意味着原本依赖 HTML renderer 在低端设备或特定 SEO 场景下的降级方案将不再可用，开发者需要评估迁移成本并调整渲染策略，同时注意字体、图片、平台 View 等方面的兼容性处理。

### [Flutter-Web 从0到部署上线（实践+埋坑）](https://zhuanlan.zhihu.com/p/1938276051617845509)
**摘要**：作者分享了将 Flutter 应用从零开始构建并成功部署到 Web 生产环境的完整实践经验。内容涵盖项目初始化、Web 特有配置、渲染模式选择、路由适配、跨域请求处理、PWA 支持开启等关键环节，并重点记录了开发过程中遇到的典型问题及解决方案。对于希望将现有 Flutter 移动应用扩展至 Web 端的团队，本文提供了宝贵的一手工程经验。

### [鸿蒙版 Flutter 环境配置 Windows 版本](https://zhuanlan.zhihu.com/p/1895965447184184217)
