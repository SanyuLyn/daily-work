## Flutter
### [Flutter Web 正式移除 HTML renderer，只支持 CanvasKit 和 SkWasm](https://zhuanlan.zhihu.com/p/11809045697)
**摘要**：官方确认 Flutter Web 正式弃用 HTML renderer，未来仅保留 CanvasKit 与 SkWasm 两种渲染方案。此举旨在统一跨端渲染表现、减少维护负担并发挥 WebAssembly 的性能优势，但也意味着首屏体积、字体加载与 SEO 等方面需要相应调整。文章提醒开发者尽早评估迁移成本，针对旧项目的渲染模式依赖提前做好适配规划。

### [Flutter-Web 从 0 到部署上线（实践+埋坑）](https://zhuanlan.zhihu.com/p/1938276051617845509)
**摘要**：面向有一定 Flutter 基础的客户端开发者，完整梳理了 Flutter Web 从项目搭建、调试到打包部署上线的全流程实践，并记录过程中遇到的典型坑点与解决方案，例如渲染模式选择、路由处理、资源加载与首屏性能优化等。文章偏工程落地，适合希望将 Flutter 能力延伸到 Web 端的团队作为实战参考。

### [Flutter Sliver 多种滚动组合开发指南](https://zhuanlan.zhihu.com/p/683463167)
**摘要**：系统讲解 Flutter 中 Sliver 体系的工作原理及多种滚动组件的组合用法，包括 CustomScrollView、SliverAppBar、SliverList 等的协同使用，帮助开发者实现复杂的嵌套滚动、吸顶 appBar、固定内容块与 tabBar 切换等交互效果，是处理高级滚动布局场景的实用技术参考。
