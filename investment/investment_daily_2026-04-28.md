## 二、当日市场关键信息

📌 一句话：五一前最后交易日，A股全线收涨，CPO光通信与半导体领涨，市场以"普涨+温和缩量"收官，等待政治局会议的节假日利好兑现。

### 2.1 宏观与政策（3 条核心观察）

- **观察 1：4月30日政治局会议"预热效应"推动节前乐观情绪**
  据华尔街见闻/财联社综合，4月30日中央政治局将召开会议，当前各大机构一致预期政策延续积极定调，"适度宽松"货币政策+财政赤字率可能超4%（≈政府借钱超过收入的4%，力度历史级别）是市场的基本预期。历史数据显示，4月政治局会议若出现超预期表述，节后首日（5月5日/5月6日）A股往往大涨。解读：节假日期间市场无法交易，但预期已充分定价，节后开盘将是压力测试。**可信度：高**

- **观察 2：货币政策"适度宽松"框架下，降息降准空间仍在**
  据第一财经/中信证券首席明明（21经济报道），2026年全年货币政策操作可能包含10BP降息（LPR = 贷款基准利率，即全国贷款的参考成本）+ 50BP降准（降低银行必须留存的准备金比例，释放资金）。当前10年期国债收益率约1.78%，仍处近年低位，资金成本偏低对股市构成支撑。解读：宽松水龙头未关，是股债双边的底层支撑逻辑。**可信度：高**

- **观察 3：人民币升值趋势延续，外资配置A股动力增强**
  据新浪财经数据，美元/人民币约6.8232，较此前6.90-6.91区间进一步升值，人民币年内累计升值幅度显著。据央行《人民币国际化报告》，明确提升融资货币功能，部分银行境外贷款上限提高。解读：人民币升值 = 外资买A股的汇兑成本降低，北向资金（外资通过沪深港通流入A股的资金，被视为"聪明钱"）持续净流入的基本面逻辑强化。**可信度：高**

### 2.2 资金面与流动性

今日资金面整体宽松。北向资金（外资通过港股通进入A股，通常被视为聪明钱的风向标）延续前期净流入态势，此前4月26日曾创历史新高，今日维持正向流入；两融余额（融资融券余额 = 市场杠杆资金总量）约2.71万亿元，维持温和扩张，显示杠杆情绪未过热。今日全市场成交约25788亿元，较前日增约1522亿元，量能温和放大，五一前"缩量收涨"格局总体健康，非"放量见顶"信号。央行MLF（中期借贷便利 = 央行向商业银行的中期贷款）本月已操作4000亿元，银行间流动性充裕，SHIBOR（上海同业拆借利率 = 银行之间借钱的成本）维持平稳低位。汇率方面，人民币兑美元6.8232，年内整体升值趋势减少资本外流压力。整体资金环境：银行资金充裕 + 外资持续流入 + 杠杆未过热，三重正向信号叠加，对权益资产友好。

### 3.2 国债 / 债券

当前10年期国债收益率约1.78%，仍在历史较低位置（1.7-1.9%是机构公认的合理区间）；30年期国债期货近期创新高，显示机构对长债的配置需求旺盛。依据：央行宽松框架维持、一季度经济数据平稳不过热（无利率上行压力）、五一期间政治局会议预期维持宽松。这周可以继续持有短债基金（1-3年久期）和纯债基金，不建议在长债高位追买，等国债收益率触及1.65%-1.70%区间再减持超长债仓位；风险提示：若5月6日后政治局会议措辞超预期强调"防风险""去杠杆"，利率可能小幅反弹，长债面临阶段性回调压力，建议届时减持10%超长债仓位转向短债。

### 3.3 股票

📌 一句话：AI算力/光通信今日领涨，绿电算电协同叙事升温，五一持股是主策略，节后等政治局会议内容再决定是否加仓。

#### 3.3.1 人工智能（AI / 算力 / 大模型）

**综合判断**：从政策看，国家支持AI基础设施建设方向明确，"算力即国力"逻辑持续强化；从景气度看，据证券时报/国金证券，全球半导体进入"全链通胀"周期，AI服务器对HBM存储芯片的需求是普通服务器数倍，供给端拐点最快要到2027年中，意味着整个2026年供需偏紧格局不变；从资金与情绪看，今日（4月28日）A股以CPO（光通信模块，AI数据中心必备的高速互联器件）、半导体为领涨板块，创业板+1.73%跑赢上证+0.52%，机构资金仍在持续增配半导体/算力ETF（交易所科创芯片类ETF资金流入持续）；从估值看，算力产业链整体估值已从传统PE（市盈率 = 股价÷每股利润）体系向PS（市销率 = 股价÷每股收入）切换，高景气给予50-60倍PE溢价，当前估值偏贵但有业绩支撑——佰维存储Q1营收同比+341%、天孚通信2026年净利润机构预期超32亿（+60%），是真实的业绩验证而非纯主题炒作。

**操作建议**：这周可以持有现有算力ETF/半导体ETF仓位（建议单一板块不超总资产15%），不建议在节前最后一天追高买入，等五一节后5月6日确认政治局会议无重大超预期内容后、若板块回调3%-5%再补仓；风险提示：短期估值偏贵+节后业绩兑现压力（Q1财报季接近尾声），若板块出现单日-5%以上放量下跌，应视为减仓信号，止损线设在持仓成本-10%。

#### 3.3.2 绿色电力（光伏 / 风电 / 储能）

**综合判断**：从政策看，据东方财富财富号综合（4月26日发布），国家层面密集出台绿电新政——工信部要求新建算力基础设施绿电使用比例≥80%（即每新建一个数据中心，80%的电要来自绿色能源），国务院明确2030年市场化交易电量占70%，2026年全国新增风光装机目标超2亿千瓦，这些是实质性的需求侧政策利好；从情绪和资金看，4月26-27日光伏ETF绝地反击，天弘光伏ETF单日净值+6.59%（据财联社统计），算电协同（绿电直连数据中心可降低算力用电成本20-30%）成为新的叙事增量，将绿电板块和AI算力的需求强行绑定；但估值分位（当前估值在历史的位置）看，年内光伏ETF整体仍亏约5%，电池ETF亏约8%，说明前期下跌已消化了大部分估值泡沫，当前PE处于历史较低位置，在政策催化下具备修复空间。

**操作建议**：这周可以小仓位（5%-8%）持有光伏/绿电ETF等待政策兑现，不建议重仓追高（4月26-27日已经大涨，短期有回踩需求），等五一后政治局会议明确"绿色转型"相关部署、或光伏ETF回踩至4月26日前水平附近（约回调3%-5%）再适度加仓；风险提示：光伏行业产能过剩问题尚未根本解决（硅料/组件价格仍在低位），政策利好更多指向运营商（绿电运营商）和储能而非纯制造端，配置方向应优先绿电运营商ETF和储能ETF而非纯光伏制造。

### 3.4 基金

当前最优品种优先级：股票型（主动科技/半导体主题ETF）> 宽基ETF（中证A500）> 黄金QDII（QDII = 合格境内机构投资者产品，让国内投资者买境外资产）> 债券型 > 货币型；AI/绿电主题ETF方向：算力/光通信ETF今日领涨，可持有不追高；绿电/光伏ETF经过两日大涨后宜观望，待回踩再布局。这周可以继续定投中证A500宽基ETF和红利ETF（市场普涨时哑铃策略两端均有回报），不建议大幅调仓（节前最后一天交易成本高、流动性略降），等节后看清政治局会议内容再决策加仓方向；风险提示：部分半导体ETF溢价率（ETF市价比净值贵的幅度）超3%，溢价追买将多付3%的隐形成本，建议查看溢价率再决定是否申购。

### 3.5 现金管理

货币基金年化约1.8-2.0%，短债基金约2.5-3.5%，银行7天理财约2.0-2.5%；七天长假期间建议将12%现金仓位放货币基金（T+1到账，不影响节后操作），可将其中1/3转入短债基金提高收益，无需提前赎回或备置大量活期存款。
